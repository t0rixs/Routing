## 詳しい説明（4000文字まで）

### 採用案（本編）

```
━━━━━━━━━━━━━━━━━━━━━━
通った道が、人生になる。
━━━━━━━━━━━━━━━━━━━━━━

歩いた道、走った線路、巡った街。
あなたの移動の一つひとつが、世界に一つだけの「あなたの地図」になっていきます。

Routepia (ルートピア) は、GPS の軌跡をヒートマップとして地図に重ねていく、極めてシンプルな記録アプリです。
華美な演出も、SNS への投稿機能もありません。
ただ、あなたの足跡が静かに地図上で増えていく――それだけのアプリです。


■ こんな方へ

・自転車やバイクで街を走り抜けるサイクリスト、ライダー
・「乗った路線」を可視化したい鉄道ファン
・休日のドライブ、車中泊、旅の記録を残したい方
・歩いて街を探索するのが好きな方
・幾何学的な美しさ、規則性のある図形に惹かれる方
・「もう二度と同じ道を通りたくない」と思ってしまう探索家気質の方


■ 主な機能

▷ ヒートマップ記録
　現在地を継続的に取得し、地図上に通った場所を色付きセルとして残します。
　通過回数が多いほど色が濃くなります。

▷ ベースマップの自由なカスタマイズ
　Light / Dark / 衛星写真 / 白紙の 4 モード。
　POI（飲食店、公園、学校、医療機関など）、鉄道駅、バス停、空港、道路ラベル、行政界ラベルを個別に表示／非表示できます。
　「自分の足跡だけを純粋に眺めたい」という方は POI を全て消して背景を白紙にすると、地図そのものが一枚の幾何学的なドローイングになります。

▷ 統計の可視化
　訪問セル数、訪問回数、累計距離などをホーム画面の HUD で常に確認できます。
　達成画面では指定領域の踏破率を見える化。

▷ ダークモード
　UI とマップを連動した完全ダークモードに対応。深夜の記録閲覧でも目に優しく、アクションシーンにも馴染みます。

▷ バックグラウンド記録
　画面を消したまま、自転車・電車・車での移動も継続して記録できます。
　バッテリー最適化済み（バックグラウンド時は描画を止め、必要最低限の DB 書き込みのみ実行）。

▷ プライバシー重視・完全ローカル
　あなたの位置情報は、あなたの端末から外に出ません。
　サーバへの送信なし、ログイン不要、アカウント登録不要。
　削除はアプリのアンインストールだけで完結します。


■ なぜヒートマップなのか

線（軌跡）ではなく「セル（マス目）」で記録するのには理由があります。
線は「いつ通ったか」を強調しますが、セルは「どれだけ通ったか」を強調します。
何度も通ううちに、自分の生活圏や行動パターンが、地図上で自然に浮かび上がってきます。

それは旅の記録というより、自分自身の縮図に近いものです。
通った道が、人生になっていく――Routepia はそのプロセスをただ静かに支えます。


■ 技術的な背景（興味のある方へ）

・GPS ログを H3 セル（六角格子）またはタイルベースで集計
・無効ピクセルの剪定とスケール変動に強いタイル合成
・低消費電力モードと前景／背景での処理切替


■ 注意事項

・GPS 機能を継続的に利用するため、純粋な機能オフ時よりは多少バッテリーを消費します（最適化済みですが、長時間記録時は充電を推奨）。
・地図表示には Google Maps を使用しています。
・本アプリはあなたの位置情報を外部送信しません。広告 SDK には Google AdMob を使用しています。

ヒートマップが、人生になる。
━━━━━━━━━━━━━━━━━━━━━━
```

### 構成のポイント

1. **冒頭の宣言** — 「通った道が、人生になる」を区切り線で囲み視覚的インパクトを出す
2. **共感パート** — 「華美な演出も SNS 投稿もない」と差別化（Strava 等との差別化）
3. **ターゲット明示** — 6 つの具体的ペルソナで自己投影しやすく
4. **機能紹介** — `▷` で視覚的に区切り。冗長にならない範囲で具体性を出す
5. **思想セクション** — 「なぜヒートマップなのか」で他のログアプリとの哲学的差別化
6. **技術背景（オプション）** — 数学・幾何学に魅力を感じる層への訴求
7. **プライバシー強調** — 完全ローカル運用は Strava や Google タイムラインと比べた強みとして
8. **クロージング** — 冒頭と呼応する三段アフォリズム
