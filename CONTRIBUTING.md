# Contributing

## OKなこと
- バグ修正、機能改善、ドキュメント改善のPull Request

## NGなこと
- 本リポジトリのコードを商用利用・再配布すること（LICENSE参照）

## 権利について
Pull Requestを送ることで、あなたは以下に同意したものとします：

- 提出した変更は、投稿者自身が作成したものであるか、
  または提出する正当な権利を有していること
- 投稿者は、提出したコードについて、
  本リポジトリ管理者に対し、
  無償かつ恒久的に利用・改変・再配布する権利を許諾します
- 当該利用には、本プロジェクトの運営および収益化
  （広告収入を含む）が含まれるものとします


## 収益化について
本プロジェクトは、広告掲載等による収益化を行う可能性があります。
Pull Request を提出することにより、投稿者は以下に同意したものとします。

- 投稿者は、提出したコードが
  本プロジェクトの運営および収益化（広告収入を含む）に
  利用されることを了承します
- 投稿者は、当該収益に対する
  報酬、分配、権利主張を行わないものとします
